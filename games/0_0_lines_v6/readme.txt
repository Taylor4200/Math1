# Lines Game V6 - Gates of Olympus Style

This is a Gates of Olympus style slot game with scatter pays, tumble mechanics, and multiplier symbols.

## Game Dimensions
- 6 reels Ã— 5 rows (30 positions total)

## Symbols
- High symbols: H1, H2, H3
- Low symbols: L1, L2, L3, L4
- Scatter: S (triggers free games)
- Multiplier: M (single multiplier symbol with weighted random values)

## Base Game
- Symbols pay anywhere on screen when 8+ of the same symbol appear (scatter pays)
- Tumble mechanics: winning symbols explode and new symbols fall in
- Multiplier symbols (M) can appear in base game
- After all tumbles, win is multiplied by the sum of all multiplier values on the board
- 3+ scatter symbols trigger 10 free spins

## Free Game (Bonus)
- 10 free spins triggered by 3+ scatters
- Retrigger: 3 scatters during free spins = +5 additional spins
- Cumulative multipliers: multiplier symbols that land add their values to a cumulative total
- Each spin's win is multiplied by the cumulative total (includes multipliers from previous spins)
- Multiplier symbols accumulate throughout the free game

## Multiplier System
- Single "M" symbol with weighted random multiplier values
- Multiplier distribution differs between base game and free game
- Frontend handles visual representation of multiplier values

## Paytable
Symbols pay based on count ranges:
- 8-9 symbols: Lower payouts
- 10-11 symbols: Medium payouts
- 12+ symbols: Higher payouts

Payout hierarchy: H1 > H2 > H3 > L1 > L2 > L3 > L4
